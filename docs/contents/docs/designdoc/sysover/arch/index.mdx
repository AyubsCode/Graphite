# 2 System Level Design

## 2.1  Hardware

<div>
<p>The system will be centered around a hardware component that functions as a central storage unit. Additionally, the system will act as a web streaming device, enabling real-time browsing and access to the data stored within it.</p>
</div>

## 2.2  Mobile Applcation 
<div>
<p>The system will feature a cross-platform application designed to ensure identical functionality and user experience across all supported operating systems. The mobile application will serve as an interface enabling users to interact with the hardware component of the system.</p>
</div>
