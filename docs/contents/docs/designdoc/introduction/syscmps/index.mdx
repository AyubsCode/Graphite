# 1.3 List of System Components

The hardware consists of a singular microcontroller , a storage unit ( microSD Card ) , and a module used to facilitate I/O operations between the MCU and the storage unit .  

The primary goal of this project is to design, develop, and deliver a fully functional cross-platform application integrated with an innovative hardware component. Together, these elements aim to address the growing need for affordable and scalable storage solutions for personal devices.
