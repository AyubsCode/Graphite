# 1.3 List of System Components

The hardware consists of a singular microcontroller , a storage unit ( microSD Card ) , and a module used to facilitate I/O operations between the MCU and the storage unit .  

The software will be developed in an Operating System Agnostic manner which will give the user complete agency to utilize whatever mobile device they choose to interface with the hardware component
